파일이름에 원치 않은 문자열이 들어갔는 경우가 있다. ex) 아이유 1집. 좋은날 - 완전고음질.mp3
'- 완전고음질' 문자열을 지우기 위해 파일이름 변경하는 툴을 찾아봤다.
